### :seedling: Repository 규칙

**BOJ/[문제이름_문제번호] 폴더에 계정아이디_src.py 저장**
>  BOJ/[문제이름_문제번호]/계정_src
>
> e.g) BOJ/[모의고사_13460]/hyeyoon0808_Main.py
>
> 함께한 문제풀이 혹은 문제리뷰 리드미로 남기기
> BOJ/[문제이름_문제번호]/README.md
